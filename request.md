# ZeroSite

帮我做一个个人网站，下面是需求

1. 网站需要有一个现代化的 UI，要求美观，支持跟随系统主题切换
2. 先完成网站首页
3. 网站使用 json 文件进行配置
4. 网站需要用于验证，账号密码通过配置文件配置，登录后才能访问其他页面，需要使用 cookies 进行登录状态管理
5. 首页有一个搜索框，接入搜索引擎，默认使用必应搜索
6. 搜索框下放置导航卡片，通过配置文件，可以配置标题，链接，图标
7. 注意代码结构管理，模块化设计，后续会增加更多功能和页面
8. 注意使用 skills，包括 premium-frontend-ui 等
9. 需要支持使用 docker compose 进行部署
10. 需要提供详细的 README.md，说明如何运行和使用该网站
